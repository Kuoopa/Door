# 基于残差随机森林的超声风速校正

本项目演示了一套面向**多源系统性误差**的超声风速数据驱动校正流程。  
与主要关注随机噪声的做法不同，本项目的仿真框架重点强调**环境层、设备层和部署层**带来的系统性偏差，并使用**残差随机森林（Residual Random Forest）**对这些误差进行补偿。

## 项目概述

基于传播时间反演的传统超声风速计算方法，在理想假设下通常具有较好的精度；但在真实应用中，由于多种系统性误差共同作用，反演结果往往会偏离真实风速。  
本项目构建了一套具有结构化误差机制的仿真数据集，并采用基于残差学习的随机森林模型进行风速校正。

校正目标定义为：

```math
residual = true\_wind - measured\_wind
```

最终校正输出为：

```math
corrected\_wind = measured\_wind + \hat{residual}
```

这种形式使模型更容易被解释为一种**误差补偿模型**，而不是一个黑箱回归器。

## 考虑的系统性误差来源

当前版本重点引入了具有明确工程意义的系统性误差，包括：

- **热膨胀 / 声程长度失配误差**
- **温度补偿模型简化误差**
- **温度代表性误差**（温度传感器测得的温度不等于声路区域真实温度）
- **安装倾角 / 风轴未对齐误差**
- **部署场景引起的局部流场畸变误差**
- **制造公差 / 设备个体差异误差**
- **双向传播通道不对称误差**
- **低成本硬件计时量化误差**
- **风速区间相关的非线性误差**
- **温度—风速耦合误差**

与单纯的高斯白噪声相比，这些误差来源更加结构化、可解释，也更适合用机器学习方法进行校正。

## 文件说明

### `app/main.py`
项目主入口文件。

主要职责：
- 加载以系统误差为主导的数据集
- 划分训练集和测试集
- 训练残差随机森林校正器
- 评估基线结果与校正结果
- 生成可视化图像

### `app/data_processing.py`
数据集构建与预处理模块。

主要职责：
- 模拟真实风速
- 注入多源系统性误差
- 生成测得传播时间和测得风速
- 构建特征矩阵
- 划分训练集和测试集

### `app/model.py`
残差随机森林模型与评估工具模块。

主要职责：
- 定义 `ResidualWindCorrector`
- 拟合残差目标
- 推理得到校正后的风速
- 计算 MAE、RMSE 和 R²

### `app/visualization.py`
分析与结果展示的可视化模块。

主要职责：
- 预测散点图
- 误差分布图
- 箱线图
- 绝对误差直方图
- 误差与特征关系图
- 特征重要性图

## 运行方式

请先确认以下文件位于 `app` 文件夹中：

```text
app/main.py
app/data_processing.py
app/model.py
app/visualization.py
```

然后运行：

```bash
python app/main.py
```

如果是在 Windows 且使用虚拟环境，也可以运行：

```bash
"D:\python project\.venv\Scripts\python.exe" "D:\python project\app\main.py"
```

## 输出内容

程序会输出：
- 基线指标（`measured_wind` 与 `true_wind` 对比）
- 校正后指标（残差随机森林输出与 `true_wind` 对比）
- MAE 和 RMSE 的提升百分比

同时会将图像保存到 `outputs/` 文件夹中，包括：
- 校正前后预测散点图
- 误差分布图
- 误差箱线图
- 绝对误差分布图
- 误差与温度关系图
- 误差与测得风速关系图
- 误差与安装倾角关系图
- 特征重要性图

## 运行结果示例

一次典型运行可能得到如下结果：

- Baseline MAE: `0.6534`
- Corrected MAE: `0.0772`
- MAE improvement: `88.18%`
- RMSE improvement: `87.50%`

这些结果表明：当前引入的误差主要属于**可学习的系统性误差**，残差随机森林能够对其进行有效补偿。

## 研究意义

本项目面向毕业设计式研究，不仅关注模型精度，也强调**误差来源分析的完整性**。

核心思路为：

1. 从超声传播物理机理出发；
2. 构建多个具有现实意义的系统误差场景；
3. 分析传统反演方法在复杂工况下的性能退化；
4. 使用随机森林学习非线性误差校正规律；
5. 证明数据驱动校准能够显著提升测量精度。

## 可扩展方向
